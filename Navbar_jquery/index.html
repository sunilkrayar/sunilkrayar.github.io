How it Works Here’s what you need to know before getting started with the navbar: Navbars require a wrapping .navbar with .navbar-expand{-sm|-md|-lg|-xl} for responsive collapsing and color scheme classes. Navbars and their contents are fluid by default. Use optional containers to limit their horizontal width.
Example Click the buttons below to show and hide another element via class changes: .collapse hides content .collapsing is applied during transitions .collapse.show shows content You can use a link with the href attribute, or a button with the data-target attribute. In both cases, the data-toggle="collapse" is required.
